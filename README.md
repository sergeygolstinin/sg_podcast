# Podcastr - AI Podcast Platform

A SaaS podcast platform with AI-powered features built with Next.js 14. Built following the JS Mastery Advanced Next.js course to learn AI integration and modern web development.

## Features

- AI text-to-speech with multiple voice options
- AI-generated podcast thumbnail images
- User authentication with Clerk
- Real-time database with Convex
- Podcast player with audio controls
- Discover and search podcasts
- User profiles and podcast management

## Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Database:** Convex
- **Authentication:** Clerk
- **AI:** OpenAI API
- **Styling:** Tailwind CSS

## What I Learned

- AI API integration with OpenAI
- Real-time database with Convex
- Authentication flows with Clerk
- Next.js App Router and server components
- Building SaaS application architecture
